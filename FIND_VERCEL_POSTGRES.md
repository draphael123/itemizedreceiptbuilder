# Where to Find Vercel Postgres

## Why It Might Not Be in That List

The list you're seeing (Neon, Supabase, Prisma Postgres, etc.) is likely the **"Integrations"** or **"Third-Party Services"** section. These are external database providers that integrate with Vercel.

**Vercel Postgres** is Vercel's **native** database service, so it's usually in a **different section**.

## Where to Find Vercel Postgres

### Option 1: Storage Tab (Most Common)

1. Go to your Vercel project dashboard
2. Look for the **"Storage"** tab (not "Integrations")
3. Click **"Storage"** â†’ **"Create Database"**
4. You should see **"Postgres"** as a Vercel-native option

### Option 2: Different Button/Location

Vercel Postgres might be accessed via:
- A **"Create Database"** button at the top
- Under **"Vercel Services"** or **"Native Services"**
- A separate **"Databases"** section
- In the project overview, look for **"Add Storage"** or **"Add Database"**

### Option 3: It Might Be Called Something Else

Look for:
- **"Postgres"** (without "Vercel" in the name)
- **"Vercel Database"**
- **"Native Postgres"**

## If You Don't See It

### Possible Reasons:

1. **Account Type**: Vercel Postgres might require a Pro plan (check your Vercel plan)
2. **Region**: Might not be available in all regions
3. **New Feature**: Might be rolling out gradually
